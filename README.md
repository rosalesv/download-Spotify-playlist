# Spotify Playlist Downloader (spotDL)

Script en Python para descargar playlists de Spotify usando **spotDL**, compatible con **Linux** y **Windows**.

El programa pide la URL de una playlist, crea automáticamente una carpeta de salida y ejecuta el comando necesario para descargar las canciones.

---

##  Características

- Descarga playlists completas de Spotify
- Compatible con Linux y Windows
- Crea automáticamente la carpeta de destino
- Uso sencillo desde terminal
- Basado en `spotDL`

---

##  Requisitos

### Común
- Python 3.8 o superior
- Conexión a internet
